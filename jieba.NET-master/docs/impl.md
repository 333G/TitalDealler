
# Newlines

对于主流的操作系统，大致划分如下：

* LF(\n): Unix(Linux, OSX, FreeBSD等)
* CR(\r): Mac OS 9及更早版本
* CRLF(\r\n): Windows, DOS

更多细节请参考[wikipedia - Newline](https://en.wikipedia.org/wiki/Newline).
